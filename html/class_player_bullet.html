Detailed Description

Obiekt pocisku gracza.

Ta klasa przechowuje tor lotu pocisku, oraz obrazenia zadane nim. Posiada tez VisibilityNotifier, aby wiadomo bylo, kiedy pocisk wyszedl poza ekran.
